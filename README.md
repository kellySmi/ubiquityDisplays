##This is a Client Server application for IoT displays


####Installation:
```bash
$ git clone https://github.com/kellySmi/ubiquityDisplays.git
$ cd ubiquityDisplays/displays-admin/static
$ npm install 
```

 		- routes.py

 	- requirements.txt
 	- run.py


 export FLASK_APP=run.py
